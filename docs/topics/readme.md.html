Version

Version comparison library for Lua. Does simple comparisons between versions, ranges and sets of ranges.

License: Apache 2.0

Copyright: Mashape Inc.

Author: Thijs Schreijer

Installation

Install through LuaRocks (luarocks install version) or from source, see the github repo.

Documentation

Can be generated using Ldoc. Just run "./ldoc ." from the repo.

Tests are in the spec folder.
